## 1. はじめに

### このマニュアルについて

このマニュアルは、**営業部の方**がYA管理システムを使うための説明書です。
難しい専門用語は使わず、画面を見ながら操作できるように書いています。

### 営業部で使える機能

| 機能 | 閲覧 | 編集 |
|------|:----:|:----:|
| ダッシュボード | OK | OK |
| トラブル対応 | OK | - |
| タスク管理 | OK | OK |
| セッション管理 | OK | OK |

**注意**: 案件（プロジェクト）の編集や、トラブルのステータス変更は**製品管理部以上**の権限が必要です。

## 2. ログイン方法

### 手順

1. ブラウザでシステムのURLにアクセスします
2. 「Googleでログイン」ボタンをクリック
3. 会社のGoogleアカウント（@yamato-mgt.com）でログイン
4. ダッシュボードが表示されれば成功です

```
ログイン画面
┌────────────────────────────────┐
│                                │
│       YA管理システム            │
│                                │
│   ┌────────────────────────┐   │
│   │  Googleでログイン  →   │   │
│   └────────────────────────┘   │
│                                │
└────────────────────────────────┘
```

### ログインできない場合

- 会社のGoogleアカウントを使っていますか？
- ブラウザのキャッシュをクリアしてみてください
- それでもダメな場合は管理者に連絡してください

### セッションについて

- **8時間**操作がないと自動的にログアウトされます
- 複数のブラウザやデバイスからログインできます
- 不審なログインがあった場合はメールで通知されます

### 各カードの説明

| カード | 内容 | クリックすると |
|--------|------|--------------|
| 今月売上 | 今月の売上合計（万円） | 損益ページへ移動 |

### クイックアクション

よく使う機能へのショートカットです。ワンクリックで移動できます。

- **トラブル登録**: 新しいトラブルを登録
- **案件一覧**: プロジェクト一覧を表示
- **未対応一覧**: 未対応トラブルだけを表示
- **タスク管理**: 自分のタスクを管理

## 4. トラブル対応

お客様や現場で発生したトラブルを記録・管理する画面です。

### 画面への行き方

1. サイドバーの「トラブル対応」をクリック
2. または、ダッシュボードの「未対応一覧」をクリック

### 一覧画面の見方

```
┌─────────────────────────────────────────────────────────────────┐
│ トラブル対応                                                    │
├─────────────────────────────────────────────────────────────────┤
│ [検索: ________] [ステータス: 全て ▼] [担当者: 全て ▼]         │
├─────────────────────────────────────────────────────────────────┤
│ 日付   │ P番号 │ 内容         │ 対応者 │ステータス│ 期限    │
│────────┼───────┼──────────────┼────────┼─────────┼─────────│
│ 02/05  │ P123  │ 設置不良     │ 山田   │ 未対応  │ 02/10   │
│ 02/04  │ P456  │ 部品交換依頼 │ 佐藤   │ 対応中  │ 02/08   │
│ 02/03  │ P789  │ クレーム対応 │ 田中   │ 完了    │ -       │
└─────────────────────────────────────────────────────────────────┘
```

### ステータスの意味

| ステータス | 意味 | 色 |
|-----------|------|-----|
| 未対応 | まだ対応が始まっていない | 赤 |
| 対応中 | 対応作業中 | 黄 |
| 保留 | 一時的に対応を止めている | グレー |
| 完了 | 対応が終わった | 緑 |

### フィルター機能

#### ステータスで絞り込み

1. 「ステータス」のドロップダウンをクリック
2. 見たいステータスを選択
   - 全て
   - 未対応
   - 対応中
   - 保留
   - 完了

#### 担当者で絞り込み

1. 「担当者」のドロップダウンをクリック
2. 担当者名を選択

#### キーワード検索

1. 検索ボックスにキーワードを入力
2. P番号、内容、会社名などで検索できます

### 詳細を見る

1. 見たいトラブルの行をクリック
2. 詳細モーダルが開きます

```
┌────────────────────────────────────────┐
│ トラブル詳細                    [×]   │
├────────────────────────────────────────┤
│                                        │
│ P番号: P123                           │
│ 日付: 2026/02/05                      │
│ ステータス: 未対応                    │
│                                        │
│ ■ トラブル内容                        │
│ 設置した製品が傾いているとのこと。    │
│ 現場確認が必要。                      │
│                                        │
│ ■ 対応内容                            │
│ （まだ記録なし）                      │
│                                        │
│ ■ 報告者: 鈴木                        │
│ ■ 対応者: 山田                        │
│ ■ 期限: 2026/02/10                    │
│                                        │
└────────────────────────────────────────┘
```

### 営業部でできること・できないこと

| 操作 | 可否 |
|------|------|
| 一覧を見る | OK |
| 詳細を見る | OK |
| フィルター・検索 | OK |
| トラブル登録 | OK |
| ステータス変更 | できません |
| 担当者変更 | できません |
| 内容の編集 | できません |
| 削除 | できません |

**ポイント**: ステータス変更や編集が必要な場合は、製品管理部の方に依頼してください。

### タスクを完了にする

1. タスクカードをクリック
2. 詳細画面で「完了にする」をクリック
3. または、タスクカードを「完了済み」エリアにドラッグ

### タスクの並び替え

- タスクカードをドラッグして好きな順番に並び替えられます
- よく使うタスクを上に置くと便利です

### 優先度の見分け方

タスクカードの左端に色付きの線が表示されます：

| 優先度 | 色 |
|--------|-----|
| 緊急 | 赤 |
| 高 | オレンジ |
| 中 | 青 |
| 低 | グレー |

## 6. よくある質問（FAQ）

### Q1. パスワードを忘れました

**A**: このシステムはGoogleアカウントでログインするため、パスワードはありません。
Googleアカウントにログインできない場合は、Googleのパスワードリセットを行うか、IT管理者に連絡してください。

---

### Q2. トラブルの対応状況を変更したいです

**A**: 営業部の権限ではステータス変更ができません。
製品管理部の方に依頼してください。

---

### Q3. 自分が登録したトラブルを削除したいです

**A**: 削除は管理部のみ可能です。
管理部に依頼してください。

---

### Q4. タスクを他の人に割り当てたいです

**A**: タスクの編集画面で「担当者」を変更できます。

1. タスクカードをクリック
2. 「編集」をクリック
3. 「担当者」を変更
4. 「保存」をクリック

---

### Q5. ダッシュボードの数字がおかしいです

**A**: データは自動更新されないため、ブラウザの更新ボタン（F5キー）を押してください。
それでも直らない場合は管理者に連絡してください。

---

### Q6. 別のブラウザでログインしたら、前のセッションはどうなりますか？

**A**: 複数のブラウザやデバイスで同時にログインできます。
ログイン中のセッションは「セッション管理」画面で確認できます。
不要なセッションは「ログアウト」で終了できます。

---

### Q7. 「権限がありません」と表示されます

**A**: その機能を使う権限がない状態です。
必要な場合は管理者に権限の付与を依頼してください。
